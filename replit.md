# Mythos Theme for Hugo/Micro.blog

## Overview
This is the Mythos theme - a minimal, fast, and highly readable Hugo theme with clean black and white design. It's customized for alfredo.design and designed for use with Micro.blog.

## Project Structure
- `layouts/` - Hugo template files (partials, singles, lists)
- `static/` - CSS, JavaScript, images, and webfonts
- `i18n/` - Internationalization files for 17 languages
- `content/` - Sample content for local preview
- `themes/mythos` - Symlink to self for Hugo development
- `config.toml` - Hugo configuration

## Running Locally
The project uses Hugo static site generator. The dev server runs on port 5000.

## Key Technologies
- Hugo (static site generator)
- CSS (Phosphor icons, custom styling)
- JavaScript (theme toggle, lightbox, avatar tilt effects)
