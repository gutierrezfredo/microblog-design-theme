# Alfredo Gutierrez Portfolio - Mythos Theme

## Overview
This is alfredo.design - a product designer portfolio built with Hugo using a customized Mythos theme. It features a minimal, clean black and white design with excellent typography.

## Project Structure
- `content/` - Blog posts, case studies, and pages (about, contact, work, etc.)
- `layouts/` - Hugo template files (partials, singles, lists)
- `static/` - CSS, JavaScript, images, and webfonts
- `i18n/` - Internationalization files for 17 languages
- `data/` - Site data files (book goals, etc.)
- `themes/mythos` - Symlink to self for Hugo development
- `config.toml` - Hugo configuration

## Running Locally
The Hugo dev server runs on port 5000 with live rebuilding.

## Key Technologies
- Hugo (static site generator)
- CSS (Phosphor icons, Font Awesome, custom styling)
- JavaScript (theme toggle, lightbox, avatar tilt effects)

## Deployment
Static site deployment via Hugo build to `public/` directory.
